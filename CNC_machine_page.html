This Page contains the SolidWorks CAD files for a Pick-And-Place (PNP) machine.

This was a project that I designed for my engineering deisgn course project. The aim of the project was to develop a low cost conceptual design for a desktop pick and place machine, to be used in an educational setting. I contributed mainly to Gantry robot design and the Frame of the PNP machine.
